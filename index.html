About

Fascinated by web development and always trying to stay up to date on the latest technologies, like Ruby on Rails, modern JavaScript, CSS, Node.js, Express, AdonisJs, Electron, React, webpack, Android, Kotlin, Laravel, and more.

Master's degree in Computer Science. Associate course degree in Software Analysis and Development, with specialization course in Software Engineering. Teacher of web and mobile development at Federal Institute of Parana (IFPR).

Have real love for Ruby on Rails since its beginning and having an intense and frequent contact with it since 2015 (teaching and as a developer). Daily working, since 2011, with app development for Android (also teaching and as a developer). Managed, developed and published Android and iOS applications, backed by Ruby on Rails servers. Eight years of previous experience with "ancient" technologies like Delphi, Java and PHP.

Trying, as far as possible, to become more involved with open source software communities. Planning to create a blog about Rails, JavaScript and all kind of stuff related to web development.

Android

Android is nowadays one of the most important mobile platforms in the world. Companies are always pursuing good Android architects and developers to create and mantain their apps. This course aims to prepare the student to be an Android developer (native platform). During its classes they learn about things like: the Android environment, the differences from other platforms like iOS, handle Android Studio and Android SDK, the Kotlin programming language, create a whole app, fetch and send data through APIs, manage multimedia inside the apps, and more.

Electron

Desktop applications are not dead yet. There are a lot of cases where we need to develop desktop client applications. However this usually implies that the team needs to choose: support only one specific platform (like Linux, Mac, or Windows) or develop two (or more) completely separated applications. Fortunately we have Electron. Electron is a framework made with Node.js that can be used for creating native desktop applications using web technologies like JavaScript, HTML, and CSS.

Frontend Web

The World Wide Web was conceived with the goal of content sharing. That makes the frontend (the part of a web application that is closer to the user) one of the most important pieces of any website. So, it's crucial for web developers to master frontend related technologies. This course intends to teach the basics of the web environment and development to the students. The topics studed during the course are HTTP protocol, HTML, CSS, JavaScript, and related techniques and technologies.

Backend Web

Not everyday a site can be totally static. In fact, they almost never can these days. Thus, we need some pieces of our applications running on the backend, inside a remote server, very frequently. We can always use any more classic approch and language to create this kind of app, like PHP, Java or Ruby on Rails, but this course explores the possibility of using JavaScript, through Node.js, Express and AdonisJs framework.
